------------------------------------------------
-- Файл с описанием аддона
------------------------------------------------
Возможности:
 - Если вводится дата с временем, то напоминание происходит единоразово. Если же указывается
   только время до события, то напоминание будет происходить каждый день
 - Выводит сообщение с временем до ближайшкго указанного события
 - Показывать или нет подсказку при наведени на кнопку "Будильник" (1-вкл/0-выкл) (задается
   в файле настроек Settings.txt)
 - Включение/отключение аддона по кнопке (ЛКМ+Shift по кнопке "Будильника")
 - Показать окно для ввода текста и времени напоминалок (ЛКМ по кнопке "Будильника")
 - Вызов окна для ввода текста и времени напоминалок с помощью чат-команды /remember
 - Добавление дополнительных полей для напоминалок в случае если все заполнены
 - Улучшен внешний вид
 - Мигание окна игры если оно свернуто за 1мин и за 5сек до события (1-вкл/0-выкл) (задается
   в файле настроек Settings.txt)

Настройки:
 - Настройка осуществляется изменением данных в файле Settings.txt. После сохранения изменений
   в файле необходимо перезагрузить аддон (Esc->Дополнения->Перезагрузить напротив Remember)

Переход на версию r3.0:
 - Удалить старую версию аддона и установить новую
 - Для переноса напоминалок:
    1. Открыть файл конфигурации /Аллоды Онлайн/data/Mods/Configs/Remember/user.cfg с помощью
       Блокнота
    2. Изменить "ScriptUserMods_REMEMBER_AddonRememberListDataGlobalTest"
       на "ScriptUserMods_REMEMBER_AddonRememberListDataGlobalTest"
    3. Изменить строки вида
        5=l" \1546386420000;val6" и 6=l"Марафон\84600000;val3"
          где ( \) и (Марафон\) - текст напоминалки или его отсутствие
              (1546386420000;) и (84600000;) - время напоминалки
              (val6) и (val3) - уникальный индентификатор напоминалки
       на строки вида
        5=l"val6; ;-1" и 6=l"val3;Марафон;84600000"
          где (val6;) и (val3;) - уникальный индентификатор напоминалки
              ( ;) и (Марафон;) - текст напоминалки или его отсутствие
              (-1) и (84600000) - время напоминалки
        Для поля val6 напоминалка отсутствует, для val3 собщение о событии "Марафон"
        ВАЖНО!!! соблюдение формы записи т.е. наличие пробела ( ) и минус один (-1) при
                 отсутствии текта напоминалки, а также количество и разделителей
                 точки с запятой (;)
    4. Сохранить изменения
